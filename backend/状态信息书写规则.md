## 状态信息书写规则

状态信息格式应书写在chart.py下的state下，所有内容使用列表包裹

+ 表格

  ```python
  [
      {
          "title": "测试1",
          "tables": {
              "cols": ["a", "b", "c"],
              "rows": [
                  [1, 2, 3],
                  [10, 20, 30],
                  [
                      { btn: "按钮", key: "这是按钮" },
                  ],
              ],
          },
      },
      {
          "title": "测试2",
          "tables": {
              "text": "你好啊"
          },
          "text": "这是一行文本",
      }
  ]
  ```

+ 文本

  ```python
  [{
  	    "text": "这是一行文本"
  }]
  ```

+ 按钮

  ```python
  [{
  	"button": {
          "btn": 按钮名称,
          "style": 按钮样式, 
          "disabled": 是否禁用, 
          "key": 按钮的唯一标签，在主程序中使用此标签调用按钮,
      }
  }]
  ```
